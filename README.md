# 🚀 Xray-Script

Простой скрипт для быстрой установки и управления Xray-core на Ubuntu.  
Автоматически настраивает протокол, получает TLS-сертификат, генерирует ссылки и QR-коды.

---

## ⚡ Быстрая установка

```bash
bash <(curl -sSL https://raw.githubusercontent.com/gkzgtzfv49-spec/Xray-Script/main/xray.sh)
```

> Требуется Ubuntu 20.04 / 22.04 / 24.04, root доступ

---

## 📋 Поддерживаемые протоколы

| Протокол | Транспорт | Описание |
|---|---|---|
| **Trojan** | WS + TLS | WebSocket поверх TLS |
| **Trojan** | TCP + TLS | Прямое подключение, низкий overhead |
| **VLESS** | WS + TLS | WebSocket поверх TLS |
| **VLESS** | TCP + TLS | Прямое подключение, низкий overhead |

## 🛠 Возможности

- **Автоустановка** — система, UFW, Xray, сертификат за один запуск
- **Автосертификат** — Let's Encrypt через acme.sh, авторенью
- **Имена пользователей** — каждый пользователь с именем, удобный список
- **Управление пользователями** — добавить, удалить, показать QR одного или всех
- **Смена протокола** — прямо из меню без переустановки сервера
- **Выбор порта** — любой порт при установке
- **WARP** — Cloudflare WARP как outbound, роутинг всего или отдельных доменов
- **Multihop** — цепочка серверов, РФ трафик напрямую, остальное через второй сервер
- **Короткая команда** — после установки просто пиши `xray-manage`

## 🔗 Multihop

Позволяет выстроить цепочку из двух серверов:

```
Клиент → Сервер 1 → Сервер 2 → Интернет
```

**Умный роутинг:**
- 🇷🇺 РФ домены и IP → Сервер 1 напрямую (быстро)
- 🌍 Остальное → Сервер 2 (через цепочку)

Оба сервера устанавливаются этим же скриптом.

## ☁️ WARP

Встроенная поддержка Cloudflare WARP:
- **Весь трафик** через WARP
- **Конкретные домены** — например `openai.com`, `netflix.com`
